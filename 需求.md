# 示例

## 概述

本项目使用 Vue3 空模板，用于演示 Vue3 的基本语法。

本项目使用 bun 管理。



## 业务

做两个页面：客户端表单填写、后台查看与编辑表单，使用 Vue Router 做跳转。

客户端页面要求实现：输入、提交、请求删除（后台同意后删除指定提交）、请求修改（后台同意后按请求修改数据）、查看提交记录

后台页面要求实现：查看、排序、处理请求

使用 JSON 保存数据，不引入数据库。



## 要求

1. 开发前查看 `package.json` 了解可用的包。

2. 页面美化放在最后一步，优先保证业务逻辑。

3. 每完成一个模块必须测试。

4. 所有功能完成后，编辑 `README.md` 说明代码结构。

5. 维护一个 `Progress.md` 记录开发过程中遇到的 bug，说明原因和解决方法。
